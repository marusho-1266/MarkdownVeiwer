プライバシーポリシー

Markdown Viewer（以下「本サイト」といいます）における、利用者情報の取り扱いについて定めます。

1. 収集する情報

本サイトは、会員登録・ログインを必要としない純粋なクライアントサイド ツールです。利用者が本サイトのエディタに入力した Markdown 本文や読み込ませたファイルの内容は、利用者のブラウザ内でのみ処理され、本サイト運営者のサーバーやデータベースへ送信・保存されることはありません。

一方で、サイトの利用状況を把握し品質を改善するため、また広告配信のために、以下の情報を Cookie 等を通じて自動的に取得することがあります。

  • 閲覧ページの URL、リファラー、滞在時間などのアクセスログ
  • IP アドレス、おおまかな地域情報
  • ブラウザ・OS・デバイスの種類、画面サイズ
  • Cookie・ローカルストレージに保存された識別子

2. ブラウザへの保存（ローカルストレージ）

本サイトは、次のような利用者の設定をブラウザのローカルストレージに保存します。これらの情報は利用者のブラウザ内にのみ存在し、運営者側には送信されません。

  • ダークモードのオン / オフ
  • 左右パネルの幅の比率
  • エディタ表示 / 非表示などの画面設定

3. Google Analytics の利用

本サイトでは、サイトの改善のために Google LLC が提供するアクセス解析サービス「Google Analytics」を利用しています。Google Analytics は Cookie を用いて、個人を特定しない形でアクセス情報を収集します。

収集されたデータは、Google のプライバシーポリシーに基づいて管理されます。利用停止を希望する場合は、ブラウザの設定から Cookie を無効化するか、Google が提供する オプトアウト アドオン を利用してください。

  • Google のプライバシーポリシー：https://policies.google.com/privacy

4. 広告配信（Google AdSense 等）

本サイトでは、将来的に Google LLC が提供する広告配信サービス「Google AdSense」を利用する場合があります。AdSense などの第三者配信事業者は、利用者の興味に応じた広告を表示するため、Cookie を使用することがあります。

  • Cookie を無効にすることで、パーソナライズ広告を無効にできます。広告設定ページ からも、Google による広告のカスタマイズを無効化できます。
  • aboutads.info の オプトアウト ページ から、多くの第三者配信事業者のパーソナライズ広告を一括で無効化できます。
  • Google による広告および Cookie の利用については、Google の広告ポリシー をご参照ください。

5. 第三者 CDN の利用

本サイトは、外部ライブラリ（Marked.js、DOMPurify、Mermaid 等）を CDN（jsDelivr 等）経由で読み込んでいます。これらの CDN 事業者は、技術的な通信に必要な範囲で IP アドレス等の情報を取得することがあります。詳細は各事業者のプライバシーポリシーをご参照ください。

6. 利用目的

取得した情報は、次の目的で利用します。

  • 本サイトの品質向上および障害・不正利用の防止
  • 利用状況の分析（Google Analytics）
  • 広告配信と効果測定（Google AdSense 等）

7. 第三者への提供

本サイトは、法令に基づく場合その他正当な理由がある場合を除き、取得した情報を第三者に提供することはありません。

8. 免責事項

本サイトに掲載された情報の正確性・有用性・安全性について、運営者は可能な限り配慮しますが、その正確性等を保証するものではありません。本サイトの利用によって生じたいかなる損害についても、運営者は一切の責任を負わないものとします。

9. ポリシーの変更

本ポリシーの内容は、法令の変更や本サイトの運用状況に応じて、予告なく変更することがあります。変更後のポリシーは本ページに掲載した時点から効力を生じます。

10. お問い合わせ

本ポリシーに関するお問い合わせは、お問い合わせページ にてご案内している方法でご連絡ください。
