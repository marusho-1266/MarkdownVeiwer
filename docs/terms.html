Markdown Viewer（以下「本サービス」といいます）の利用にあたっては、以下の規約に同意いただいたものとみなします。

  5. 不正アクセスを試みる行為、またはセキュリティを侵害する行為

第4条（利用者が作成したコンテンツ）

利用者が本サービスを使って作成した Markdown 本文および生成物（プレビュー HTML、PDF 等）の著作権は利用者または正当な権利者に帰属します。運営者は当該コンテンツをサーバーに保存・送信しないため、その内容に関して運営者は一切関知せず、責任も負いません。
