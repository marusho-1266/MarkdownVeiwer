Markdown Viewer — ヘルプ

左で編集・右でプレビューできるシンプルな Markdown ビューアの使い方です。

基本の流れ

  1. 左のエディタに Markdown を入力すると、右にプレビューが表示されます。
  2. ローカルの .md / .markdown / .txt を左の入力エリアにドラッグ＆ドロップすると内容を読み込めます。エディタを折りたたんだときは、左の余白（「エディタを表示」付近）にドロップしても同様に読み込めます。
  3. 上部の印刷から、プレビュー内容を印刷できます（印刷ダイアログでプレビュー確認を推奨）。

画面上部のボタン・設定

  • エディタを隠す / 表示：左パネルを折りたたみ、プレビューを広く表示します。折りたたみ中は画面左端の「エディタを表示」からも開けます。
  • ダークモード：配色の切り替え（設定はブラウザに保存されます）。
  • ヘッダーに印刷時間を表示：印刷時のヘッダーに日時を出すかどうか。
  • スクロールを連動：オンにすると、左エディタと右プレビューの縦スクロール位置がおおよそ連動します。

Mermaid 図表

コードブロックの言語に mermaid を指定すると図として描画されます。

```mermaid
flowchart LR
    A --> B
```

エディタの右クリック

テキストエリア上で右クリックすると、切り取り・コピー・貼り付け、改ページを挿入（印刷時の改ページ用）、エディタを隠す／表示が選べます。

左右の幅

左パネルとプレビューの間の縦線をドラッグすると、幅の比率を変更できます（比率は保存されます）。
