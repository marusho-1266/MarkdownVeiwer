使い方ガイド

ここでは Markdown Viewer の詳細な使い方と、よく使う Markdown 記法・Mermaid 図表の書き方・印刷テクニックをまとめています。

1. 画面構成

  • 上部ヘッダー：印刷・エディタ表示切替・ダークモード・使い方（本ガイドを別ウィンドウで開く）・印刷時刻のオン/オフ・スクロール連動など。
  • 左パネル（エディタ）：Markdown を入力する領域。行番号付きのテキストエリアです。
  • 中央リサイザー：縦線をドラッグすると左右パネルの幅を調整できます（設定はブラウザに保存）。
  • 右パネル（プレビュー）：入力した Markdown をリアルタイムに HTML 化して表示します。

2. ファイルの読み込み方

  1. PC のファイラから .md、.markdown、.txt のいずれかを選ぶ。
  2. エディタ領域またはプレビュー領域へドラッグ＆ドロップする。
  3. 既存のエディタ内容が置き換えられ、プレビューが更新されます。
補足：複数ファイルを一度にドロップした場合は、先頭の 1 件のみが読み込まれます。ローカル処理のため、ファイルの内容はネットワーク上に送信されません。

4. Mermaid 図表

コードブロックの言語指定を mermaid にすると、図として描画されます。

```mermaid
flowchart LR
    A[開始] --> B{条件}
    B -- Yes --> C[処理A]
    B -- No  --> D[処理B]
    C --> E[終了]
    D --> E
```

フローチャートのほか、シーケンス図（sequenceDiagram）、ガントチャート（gantt）、クラス図（classDiagram）などにも対応します。詳細は Mermaid 公式ドキュメント をご参照ください。

5. 印刷 / PDF への出力

  1. ヘッダー右側の「印刷」ボタンを押します。
  2. ブラウザの印刷ダイアログで、プリンターの代わりに「PDF として保存」を選ぶと PDF を生成できます。
  3. 「ヘッダーに印刷時間を表示」をオンにすると、印刷時の右上に日時が入ります。
  4. フッターには 現在ページ / 総ページ数 が自動で入ります。

改ページを指定したい

エディタ上で右クリックして「改ページを挿入」を選ぶと、その位置にカーソルがある行の先頭へ改ページ用の HTML が挿入されます。印刷時にその位置で必ず改ページされます。

7. よくあるご質問

Q. 入力した Markdown はサーバーに送信されますか？

A. いいえ。プレビュー変換・印刷処理はすべてお使いのブラウザ内で完結します。入力した本文がサーバーへアップロードされることはありません。

Q. スマートフォンでも使えますか？

A. 表示は可能ですが、エディタと左右パネル構成のため画面サイズの大きい PC・タブレットを推奨します。

Q. 印刷時に、左のエディタも一緒に印刷されてしまいます

A. 本アプリは、印刷時にエディタ側を非表示にする CSS を適用しています。ブラウザの印刷設定で「背景グラフィック」「ヘッダーとフッター」を適宜切り替えると意図通りのレイアウトになりやすいです。

Q. 対応しない Markdown 記法はありますか？

A. 基本的な Markdown 記法には対応していますが、一部の拡張記法（脚注、MathJax による数式など）は未対応の場合があります。
