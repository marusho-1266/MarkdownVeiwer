お問い合わせ

Markdown Viewer をご利用いただきありがとうございます。ご意見・ご要望・不具合のご報告は、下記の方法にてお寄せください。

GitHub Issues（推奨）

バグ報告や機能要望は、GitHub リポジトリの Issues からご連絡いただくのが最も確実です。再現手順やスクリーンショットがあると原因特定がスムーズです。

  • リポジトリ: github.com/marusho-1266/MarkdownVeiwer
  • Issues: github.com/marusho-1266/MarkdownVeiwer/issues

お問い合わせいただく前に

  • 使い方ガイド に、操作方法・Markdown 記法・よくある質問を掲載しています。あらかじめご確認ください。
  • 不具合の場合は、お使いのブラウザの種類とバージョン、発生した手順、期待した挙動と実際の挙動を添えてご連絡ください。
  • 入力した Markdown 本文そのものを共有する必要はありません（本サービスは本文をサーバーに送信しません）。差し支えない範囲で、再現する最小限のサンプルを添えていただけると助かります。

広告・プライバシーに関するお問い合わせ

広告掲載やプライバシーポリシーに関するご質問・ご指摘についても、上記 Issues からご連絡いただけます。

ご注意：個別の技術サポートや応答をお約束するものではありません。いただいたご意見は今後の改善の参考とさせていただきます。
