このサイトについて

Markdown Viewer は、ブラウザだけで Markdown を書き・見て・印刷／PDF 化できる、インストール不要の軽量ツールです。

コンセプト

議事録・仕様書・手順書・学習メモなど、日々の業務や学習で発生する「ちょっとした Markdown 文書」を、 インストール不要・ログイン不要で素早く整形して印刷／PDF 化することを目的としています。 ブラウザのタブをひとつ開くだけで使える点を最優先に、余分な機能は持たない設計としています。

主な特徴

  • リアルタイムプレビュー：左側のエディタで入力した Markdown が、右側に即座に反映されます。
  • ドラッグ＆ドロップ読み込み：ローカルの .md / .markdown / .txt ファイルを画面に落とすだけで読み込めます。
  • Mermaid 図表：フローチャート、シーケンス図、ガントチャートなどを Markdown に直接書いて描画できます。
  • 印刷／PDF 出力：プレビューだけを対象に、ページ番号・印刷時刻・改ページを指定して印刷できます。ブラウザの「PDF として保存」でそのまま PDF 化できます。
  • ダークモード：OS のテーマとは独立して切り替え可能。設定はブラウザに保存されます。
  • オフラインで完結する編集：入力した Markdown の内容はサーバーに送信されません。処理はすべてブラウザ上で完結します。

想定される利用シーン

  • 社内向けの簡易な手順書・仕様書の清書と印刷
  • ChatGPTなどの生成AIが返した Markdown 形式の回答を、そのまま貼り付けてプレビュー・軽微な手直し・印刷／PDF 化する（追加の導入物なしで、ブラウザだけで完結します）
  • 議事録や学習ノートを整形して PDF で保管
  • GitHub や Qiita 等に貼る前の Markdown の整形確認
  • Mermaid による図表のプレビュー

技術スタック

  • Markdown パーサー: Marked.js
  • HTML サニタイズ: DOMPurify
  • 図表: Mermaid
  • フロント: 素の HTML / CSS / JavaScript（単一ファイル構成）
  • ホスティング: Render の静的サイト

データの取り扱い

編集中の Markdown テキストはブラウザ内でのみ処理され、サーバーへアップロードされません。 アクセス解析のため Google Analytics を利用しているほか、広告配信のため Google AdSense を利用する場合があります。 詳細は プライバシーポリシー をご覧ください。
